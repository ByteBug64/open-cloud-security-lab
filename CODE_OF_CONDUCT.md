# CODE OF CONDUCT

## Our Pledge
We are committed to providing a welcoming, inclusive, and respectful environment for everyone.

## Expected Behavior
- Be respectful and kind
- Give constructive feedback
- Be patient with beginners
- Collaborate openly

## Unacceptable Behavior
- Harassment or discrimination
- Personal attacks
- Sharing private or sensitive information
- Disruptive or toxic behavior

## Enforcement
Project maintainers have the right to remove content or contributors that violate this code of conduct.

## Scope
This code applies to:
- GitHub issues
- Pull requests
- Discussions
- Any project-related communication
